# Projet Album de musique

## Description du projet
Ce projet consiste en la création d'un **album de musique** où l'on devait intégrer les éléments suivants    :
- **TITRE DE L'ALBUM**
- **NOM DE L'ARTISTE**
- **TRACKLIST**
- **DISCOGRAPHIE**

L'objectif était également d'en faire une **page web** hébergée via **GitHub Pages**.
**[Voir le site en ligne](https://robertandresvallejos.github.io/ALBUM_LA_TOILE/)**

## Langages utilisés
- **HTML** 
- **CSS**
